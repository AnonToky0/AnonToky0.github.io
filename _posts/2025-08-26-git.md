### 常见的提交类型示例

| 类型    | 含义                            |
|---------|--------------------------------|
| feat    | 新功能                         |
| fix     | 修复 bug                      |
| docs    | 文档修改                      |
| style   | 代码格式（不影响功能）          |
| refactor| 重构（不新增功能或修复 bug）    |
| test    | 添加或修改测试                 |
| chore   | 其他杂务（构建过程或辅助工具变动） |
